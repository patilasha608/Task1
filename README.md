## 📝 Description
This project is a responsive frontend web application built using **HTML, CSS, and JavaScript**.

It includes:

* A modern Login Page
* A Signup Page
* A Professional Dashboard Layout
* Responsive design for Mobile, Tablet, and Desktop

The dashboard layout includes:

* Sidebar navigation
* Header section
* Content area
* Responsive card layout

This project demonstrates frontend development skills, UI structuring, and responsive design principles.

git clone <your-repository-link>

  OR

---

### Step 3 — Run the Project

You can run it in two ways:

#### Option 1: Direct Open

Double click:

```
index.html
```

It will open in your browser.

#### Option 2: Using Live Server (Recommended)

1. Install Live Server extension in VS Code
2. Right-click on `index.html`
3. Click **Open with Live Server**
